Welcome

This portal now mirrors the project spec workflows. Use the navigation above to move between the focused views instead of a single overwhelming page.

  • Manage Degrees – create degrees, tie courses to each degree, and map objectives per course.
  • Manage Courses – keep the course catalog and instructor roster accurate.
  • Manage Objectives – define learning objectives with short codes and unique titles.
  • Manage Semesters & Sections – add semesters, sections, and assign instructors with enrollment totals.
  • Associate Courses with Objectives – link any catalog course to the objectives it supports (independent of degrees).
  • Enter/Review Evaluations – drive the per-section evaluation workflow with summaries and completion status.
  • Run Queries / Reports – execute the required spec queries with SQL-side filtering and chronological ordering.

Work through the tabs from left to right when onboarding a new database: define master data, schedule sections, connect objectives, then enter evaluations and run reports.
