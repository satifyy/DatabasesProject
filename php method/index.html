Curriculum Assessment Toolkit

This project combines a MySQL schema, a PHP web interface, and Python utilities to manage degrees, courses, objectives, sections, and instructor evaluations with complete CRUD + reporting workflows.

Start the Web App

Launch PHP's built-in server from the repository root:

php -S localhost:8000 app.php

Then open the application:

Open Curriculum Portal

Provision the Database

  1. Edit config.ini with your MySQL credentials.
  2. Initialize schema + demo data:
    python main.py --init --seed
  3. Use programfile.py for scripted reports:
    python programfile.py demo

Project Highlights

  • Natural keys + cascading foreign keys, enforced checks, and constraints directly from the ER model.
  • Full CRUD for degrees, courses, instructors, semesters, objectives, and section entry.
  • Guided degree-to-course and course-to-objective workflows with guard rails for core courses.
  • Evaluation cockpit that enforces per-objective tracking, partial/complete states, and duplication across degrees.
  • Built-in reporting widgets plus Python CLI queries for reproducible demos.
